## Simple Chat Application

### Description:

Simple chat application made in Typescript using Vite React Nodejs and Stream and Fastify.

NOTE:- Stream API is used which is only free for 30days.

### Technologies Used:

#### Client:

- Vite
- React, React Dom
- React Router Dom
- Tailwind
- Axios
- React Query
- Stream Chat

#### Server:

- Node
- Fastify
- Nodemon
- Dotenv
- Stream Chat
